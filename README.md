# Healthcare Waiting List Analysis Dashboard

This project showcases an interactive **Power BI dashboard** built to analyze healthcare waiting list data and uncover trends across time, age groups, case types, and medical specialties.

## Overview
The dashboard provides a high-level summary of waiting list performance, enabling quick comparison between the **latest month** and the **previous year**, as well as deeper insights into average and median waiting times.

## Key Insights
- Latest Month vs Previous Year waiting list comparison
- Average and Median waiting time analysis
- Breakdown by Case Type (Outpatient, Day Case, Inpatient)
- Waiting list distribution by Age Profile and Time Bands
- Trend analysis over time using archive dates

## Tools & Technologies
- Power BI  
- DAX (measures and calculations)  
- Data modeling and transformation

## Dashboard Pages
- **Summary Page**: Executive-level overview with KPIs and trends  
- **Detail Page**: Granular analysis by specialty, age group, and case type
